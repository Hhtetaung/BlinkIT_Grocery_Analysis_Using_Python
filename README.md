# BlinkIT_Grocery_Analysis_Using_Python
This project is a Python-based data analysis of the BlinkIT Grocery Store dataset. It builds on the same dataset used in my previous analysis, but this time the entire workflow is completed using Python. The project covers data loading, data cleaning, KPI calculations, and exploratory data analysis (EDA) to uncover sales trends and business insights. Various visualizations are created using Matplotlib and Seaborn to analyze metrics such as total sales by item type, outlet size, outlet location, establishment year, and fat content. The goal of this project is to demonstrate practical data analysis skills using Pandas, NumPy, Matplotlib, and Seaborn while transforming raw data into meaningful business insights.
